1 - Nome do projeto
LeiloesTDSat (Sistema de Leilões)

2 - Sobre o projeto
Aplicação desktop desenvolvida em Java Swing para gerenciamento de um sistema de leilões/vendas de produtos.
O sistema permite cadastrar novos produtos (nome e valor) através de uma tela de cadastro, e visualizar a listagem de produtos cadastrados em uma tela própria,
exibindo informações como ID, nome, valor e status (por exemplo, "A Venda" ou "Vendido"). 
A tela de listagem também conta com uma funcionalidade para marcar um produto como vendido a partir do seu ID. O projeto segue o padrão DAO/DTO para separar a lógica de acesso a 
dados (persistência em banco MySQL) das entidades de domínio, e foi desenvolvido no NetBeans como parte de uma atividade acadêmica (UC11).

3- Tecnologias utilizadas
Java (Swing para interface gráfica)
MySQL (banco de dados uc11, tabela produtos)
JDBC (conexão Java–MySQL via DriverManager)
NetBeans IDE (ambiente de desenvolvimento e build via Ant)
Padrão DAO/DTO (ProdutosDAO, ProdutosDTO, conectaDAO)
